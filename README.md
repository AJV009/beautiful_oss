# BOSS: Beautiful Open Source Software Blog.

## To connect DB ->
1. Create a cred.json file with details like.
```json
{
    "host":"host_name_or_ip",
    "user":"DB_username",
    "pass":"DB_password"
}
```
or you can specify a custom json file in [sqlmanager.php](php/sqlmanager.php)
